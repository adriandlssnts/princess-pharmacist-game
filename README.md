# 👑 Princess Pharmacist Game

A cute, interactive pharmacy learning game with real clinical concepts.

## 💊 Features
- Drug selection
- Dose & frequency logic
- Pediatric dosing
- Clinical decision-making

## 🎮 How to Play
1. Start a case
2. Choose:
   - Drug
   - Dose
   - Frequency
3. Submit treatment

## ⚠️ Disclaimer
This game is for educational purposes only and not medical advice.
